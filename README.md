# 操作逻辑

🌸 启动项目后，会弹出一个小窗窗～
🍀 先手动打开【QQ经典农场】小程序
👉 然后戳一戳 【自动拾取】（小助手会自动找到小程序在屏幕上的位置～不过每次移动窗口后，都要重新点一下拾取喔💡）
✅ 等窗口拾取成功，就可以开心地点 【启动任务】 啦！

⚙️ 如果想自己安排任务，可以点下面这两个小按钮：
🌾 主界面任务 和 👭 好友农场任务
✨ 想做什么就勾什么，立刻生效～

## 环境

- Windows 10/11  
- Python 3.10+（建议 3.11）

## 安装依赖

在项目根目录执行：

```bash
py -3 -m pip install -r requirements.txt
```

没有 `py` 时用：`python -m pip install -r requirements.txt`

## 启动

双击 **`启动.bat`**，或在项目根执行：

```bash
python gui_scripts/new_main_pyqt.py
```
