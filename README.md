# QQ 农场控制台（qq-farm-vt）

Windows 下图形控制台：游戏窗口识别、任务控制、种植策略参考。仅供学习与交流，请遵守游戏服务条款。

# 项目介绍
本项目为ai辅助项目，可以确保主界面收获、除草除虫以及好友界面偷菜除草除虫功能没有问题，
种植功能尚未开发，
可以确保不被封号，纯视觉操作，不卡

# 操作逻辑
启动项目会弹出界面
需要你先打开qq经典农场小程序，点击自动拾取，脚本会自动判定小程序在屏幕中的位置。每次移动需要重新拾取
确认窗口拾取后，即可点击启动任务，

如果你有需要设置的，可以点击下方的主界面任务，和好友农场任务，勾选即可生效。

如果使用过程中遇到问题，可以留言~

## 环境

- Windows 10/11  
- Python 3.10+（建议 3.11）

## 安装依赖

在项目根目录执行：

```bash
py -3 -m pip install -r requirements.txt
```

没有 `py` 时用：`python -m pip install -r requirements.txt`

## 启动

双击 **`启动.bat`**，或在项目根执行：

```bash
python gui_scripts/new_main_pyqt.py
```
